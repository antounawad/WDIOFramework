2. Verzeichnis Automatic_Test anlegen
	
3. Installation phantomjs
	npm install phantomjs
	
	
4. Install Selenium standalone server
	http://selenium-release.storage.googleapis.com/index.html?path=3.4/
	Zum Starten des Selenium Servers:
	Java		Gewünschter Treiber  mit Pfat zur Binärdatei
	java -jar -Dphantomjs.binary.path=C:\Automatic_Test\node_modules\phantomjs\lib\phantom\bin\phantomjs.exe selenium-server-standalone-3.4.0.jar


5. Installation webdriverio
   npm install webdriverio >> Required
   
6. Testfile erstellen
   und in Root Verzeichnis speichern.
   
   
   var webdriverio = require('webdriverio');
	var options = {
    desiredCapabilities: {
        browserName: 'firefox'
    }
	};
webdriverio
    .remote(options)
    .init()
    .url('http://www.google.com')
